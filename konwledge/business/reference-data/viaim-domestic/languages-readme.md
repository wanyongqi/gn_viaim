# 国内版 viaim 支持语种说明

> 数据源：`languages.csv`（拷贝自 `docs/source-data/软件支持语种 - viaim 国内版本.csv`，保持同步）
>
> 版本：V1.0.0 | 最后更新：2026-04-20

---

## 一、字段说明

| 列 | 含义 |
|---|------|
| 语种 | 语种名称（中文） |
| 三种录音模式 | 通话/现场/音视频三种录音的转写支持 |
| 同传听译 | 实时翻译与录音（手机麦克风收音 + 转写+翻译+TTS）的源/目标语言支持 |
| 面对面翻译 | 国内专属功能；两端互译翻译场景 |
| 通话翻译 | 国内专属功能；通话过程中实时翻译 |
| 离线翻译 | 无网环境下上述翻译功能支持的语种（仅中文、英语）|
| 优先级 | 测试覆盖优先级（P0 最高，P4 最低），用于语种抽样测试 |

---

| 同传听译 | **35 种** | 32 种外语 + 粤语 + 2 种民族语言（藏语/维吾尔语）|

| 离线翻译 | **2 种** | 仅中文 / 英语 |

---

## 四、特殊语种说明

| 语种 | 特点 |
|------|------|
| 方言（12 种）| **仅三种录音模式**支持，**不支持**同传/面对面/通话翻译 |
| 藏语、维吾尔语 | 支持三种录音、同传听译、通话翻译；**不支持面对面翻译** |
| 粤语 | 全部翻译功能支持（同传/面对面/通话）；也支持三种录音 |
| 中文、英语 | P0，**唯二支持离线翻译**的语种 |

## 五、引用指引

### 写转写语种测试用例
- 引用本 CSV 的"三种录音模式"列
- 覆盖策略：P0 全覆盖 / P1-P2 各 1 例 / P3-P4 抽样 / 方言至少 1 例 / 民族语言 1 例

### 写翻译语种测试用例
- 同传听译：引用"同传听译"列（35 种）
- 面对面翻译：引用"面对面翻译"列（33 种）
- 通话翻译：引用"通话翻译"列（35 种）

### 写离线场景测试用例
- 无网 / 弱网下，翻译功能**仅中英互译可用**，其他语种应置灰或提示
- 涉及文件：`features/viaim-domestic/translation/*.md`
