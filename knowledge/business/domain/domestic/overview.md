# 国内版 viaim App 全局专属规则

> **本文件为国内版测试用例生成的必读文档**。列出所有与海外版有本质差异的全局规则。生成国内版用例时，如发现海外版 domain/features 文档与本文件冲突，**以本文件为准**。
>
> 版本：V2.4.0 | 最后更新：2026-04

## 一、与海外版的根本性差异总览

| 维度 | 海外版 viaim | 国内版 viaim |
|------|-------------|-------------|
| 套餐层级 | Basic / Pro / Ultra + 未激活会员 | **无套餐**（注册即用全部功能）|
| 转写时长额度 | 按套餐分级（如 Basic 300 分钟/月）| **无时长额度限制**；但**非实时转写**（详情页"音频转文字"/"重新转写"）对单条录音**限制 2 次**，第 3 次起按钮消失（见 `detail-page.md` §8.1）|
| AI 模型权限 | 免费/付费模型按套餐开放 | **全部可用**；使用国内合规模型（具体清单待确认）|
| Space（App 端）| 支持，独立存放音频/文档 | **无独立 Space 菜单**，但支持在录音列表中创建**自定义分组**（见 `record-list.md` §1.1） |
| Space（Web 端）| 支持 | 改称「分组」，语义更弱（见 `features/viaim-domestic-web/space-grouping.md`）|
| 上传功能（App 端）| 支持音频上传 | **不支持**，仅通过录音/闪录产生记录 |
| 上传功能（Web 端）| 支持音频 + 文档 | ⚠️ 待确认国内 Web 上传范围（见 `features/viaim-domestic-web/upload-and-capacity.md`）|
| 文档类型记录 | 支持（带文档详情页）| **不支持**，仅音频类型 |
| 主题模式 | 浅色 / 深色 / 跟随系统 | **仅深色模式** |
| 登录方式 | 邮箱 / Google / Apple / 手机号 | **仅手机号+短信验证码 + App 扫码** |
| 账号绑定 | 邮箱/手机/Google/Apple 多种绑定 | **手机号为唯一主登录**，不支持解绑，无三方绑定，无密码体系 |
| AI 助理名称 | Vitana（固定）| **按界面语言切换**：中文→万姆 / 英文→Vitana |
| AI 助理入口 | 底部 Tab + 详情页按钮 | **无底部 Tab**；入口仅 ① 耳机语音唤醒「你好，小万/Hey Vitana」（仅 Pro3/Air2 + 开关开启）② 录音详情页右下角浮球 |
| 语音控制指令语言 | 英文 | **中文**（「播放音乐/暂停播放/上一首/下一首/增大音量/减小音量/接听电话/挂掉电话/降噪模式/通透模式/开始录音/停止录音」）；随耳机语言设置切换 |
| 语音控制型号支持 | 同海外 | **语音控制**：Pro3/Air2/Pro2 支持；Nano+/VIAIM Air 不支持 <br>**语音唤醒万姆**：仅 **Pro3/Air2**；Pro2/Nano+/VIAIM Air 设置页不展示开关 |
| 转写语言 | 145 种（含地区变体）| **46 种**（32 外语 + 12 方言 + 藏/维吾尔 2 民族语言）|
| 翻译语言 | 80 种 | **同传听译 35 种 / 面对面翻译 33 种 / 通话翻译 35 种** |
| 翻译功能数 | 1（同传听译）| **3 个**（同传听译 + 面对面翻译 + 通话翻译）|
| 离线翻译 | 不支持 | **支持，但仅中英两种**（用于无网场景）|
| UI 界面语言 | 多语言 | 仅简体中文 / English（与上面"转写/翻译"区分）|
| 客服渠道 | Fin.ai | **国内客服**（具体渠道名称待确认）|
| 隐私政策 | GDPR / CCPA 等 | **PIPL 合规中文版** |
| 底部导航 | 3 Tab（记录 / 空间 / 我的）| **4 Tab：首页 / 录音记录 / 发现 / 我的** |
| 新记录来源 | 录音 / 闪录下载 / 上传音频 / 上传文档 | **仅录音 / 闪录下载** |
| 云数据存在方式 | 固定上云 | **由「云同步」开关控制**（位于「账号设置」中，见 `account-settings.md` §1.3）|
| 版本迁移 | 分组→空间迁移 + 祖父条款 | ⚠️ 国内版是否有老版本迁移需求待确认 |

## 二、核心差异的连带影响

以下连带行为在国内版 features 文档里已分散体现，**生成用例时无需逐条校验，但需整体遵守**：

### 2.1 无独立 Space 菜单连带（改为自定义分组）
- 记录列表顶部：支持下拉切换「全部记录/回收站/分组」（见 `record-list.md` §1.1）
- 左滑快捷操作：**无「添加到空间」**，只有删除
- 批量操作按钮：**合并 / 移到 / 导出 / 删除**（「移到」指移到分组）
- 合并录音规则沿用 V6.3.0：前置校验优先级、上限（10 条 / 5 小时 / 每日 100 次）、拖拽排序、标题递增、保留原音频开关、结果拼接与 AI 重生成（见 `record-list.md` §4.1~§4.5）
- 详情页更多菜单：**无「移除」操作**
- AI 助理引用范围：支持「当前记录 / 最近记录 / **我的分组**」
- 搜索范围：**无 Space 维度**
- 筛选维度：**无「音频位置（已加入空间/未加入空间）」**

### 2.2 无上传连带
- 记录列表右上角：**无「导入/添加记录」按钮**
- 搜索结果类型：**无「文档」筛选项**（国内版无文档类型记录）
- 详情页类型：**仅两种**（有转写的音频 / 仅录音），**无文档详情页**
- 批量导出：**无「导出原文件」选项**

### 2.3 无套餐连带
- 不出现「升级弹窗」「剩余时长」等文案
- 不出现「未激活会员」「设备绑定激活 Basic」等流程
- 不出现「多语言付费」提示
- AI 模型选择不出现「付费模型」置灰或付费引导
- **转写额度**：国内**不存在**"实时/非实时转写时长额度"概念；但**非实时转写对单条录音有 2 次上限**（基于次数而非时长，见 `detail-page.md` §8.1）——这是国内独有的限制形态，**非来自套餐/额度**

### 2.4 语种与翻译连带
- **UI 界面语言**：仅简体中文 / English
- **转写语言**：46 种（32 外语 + 12 方言 + 2 民族语言，详见 `reference-data/viaim-domestic/languages.csv`）
- **翻译功能**：3 个（同传听译 / 面对面翻译 / 通话翻译）
- **翻译语种范围**：同传 35 种 / 面对面 33 种 / 通话 35 种
- **离线翻译**：无网环境下，3 种翻译功能均只支持**中英互译**
- **方言不可翻译**：方言仅用于转写，不出现在任何翻译语言列表
- **藏语/维吾尔语**：支持转写、同传、通话翻译；**不支持面对面翻译**
- 实时翻译与录音（= 同传听译）首次启动默认语种：
  - 界面中文 → 转写 英语 / 翻译 中文
  - 界面英语 → 转写 西班牙语 / 翻译 英语（西班牙语在国内版语言列表中存在，优先级 P2）

### 2.5 AI 助理双名称连带
- 国内版**无底部 AI 助理 Tab**（海外特性）；入口仅 ① 耳机语音唤醒（Pro3/Air2 开关开启时）② 详情页右下角浮球
- 唤醒词：中文=「你好，小万」 / 英文=「Hey Vitana」（⚠️ 英文原文待确认）
- 页面标题：中文=万姆 / 英文=Vitana
- 详情页 AI 浮球图标/tooltip 名称同步切换
- AI 设置 → 聊天机器人模型 → 同步到万姆/Vitana
- 切换界面语言后全部 UI 同步更新，但**不影响** AI 输出语言设置
- **AI 引用范围**：当前记录 / 最近记录 / 我的分组（用户在录音列表自建的分组，见 `wanmu-vitana.md` §四）

## 五、测试关注点（跨模块）

- **平台识别**：生成用例前务必确认测试对象是国内 App / 海外 App / 国内 Web / 海外 Web，避免误引用海外 domain 规则
- **无套餐场景回归**：涉及 AI 功能、转写、录音的用例，不应出现"超出额度""升级弹窗"等海外文案；但**非实时转写 2 次上限**是国内独有规则，必须覆盖
- **无独立 Space 场景回归**：列表/详情页/搜索等不出现 Space 文案是已确认的；但国内版支持**自定义分组**（在录音列表顶部切换、批量操作中有「移到」分组、AI 助理引用范围中有「我的分组」）——用例需覆盖这些分组相关交互
- **云同步开/关切换**：是跨模块的核心变量，详见 `account-settings.md` §1.3
- **AI 助理名称跟随**：切换界面语言后全部 UI 入口同步变更（万姆 ↔ Vitana）
- **仅中英 UI 场景**：界面只切换中/英，但**转写/翻译语种列表仍多语言**，用例不要误以为转写也只有中英
- **无多语言付费**：转写/翻译语言选择不出现"付费升级""多语言付费"等海外文案（国内无套餐）
- **新老翻译功能区分**：同传听译/面对面翻译/通话翻译三者独立，语种列表不同、UI 不同、记录归属不同
