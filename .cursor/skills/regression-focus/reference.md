# 测试回归重点建议参考清单

## 回归优先级判定标准

### 必回归（P0）
满足以下任一条件：
- 本次改动直接修改了该功能的代码路径或交互逻辑。
- 属于核心主流程强感知功能（录音启停、转写上屏、录音保存、单条导出）。
- 上次版本该功能曾出现 Bug，本次改动触及相关路径。

### 建议回归（P1）
满足以下任一条件：
- 改动触及公共组件或状态机，可能间接影响该功能。
- 该功能与改动点存在共享逻辑（如同一弹窗、同一语言选择器）。
- 改动点涉及数据结构或存储格式，历史记录可能受影响。

### 可选回归（P2）
- 改动与该功能无明显关联。
- 该功能为低频边缘场景，影响用户面窄。
- 按测试资源决定是否覆盖。

## viaim 核心主流程（每版必回归）

| 功能 | 验证重点 |
|------|----------|
| 通话/现场/音视频录音启停 | 正常开启、暂停、恢复、结束保存 |
| 实时转写上屏 | 转写正确上屏，无丢字、卡顿 |
| 录音保存与详情页 | 生成记录，音频/转写/AI内容可正常查看 |
| 单条导出 | 音频/转写导出格式正确，无乱码 |
| 蓝牙耳机连接 | 连接/断连/重连状态正常 |

---

## 常见间接影响路径

| 改动类型 | 可能影响的功能 |
|----------|---------------|
| 录音状态机变更 | 暂停/继续、耳机控制、切后台恢复 |
| 语言选择逻辑变更 | 转写语言、翻译语言、语种记忆 |
| 列表/详情页公共组件 | 搜索、导出、标记、合并录音 |
| 导出链路变更 | 单条导出、批量导出、zip 打包 |
| AI 服务接口变更 | 智能标题、摘要、脑图 |
| 存储格式/数据结构变更 | 老数据兼容性、历史录音查看 |
